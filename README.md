# Lappiz Demo Frontend

Frontend del MVP de registro de interesados para la demo de Lappiz LowCode.  
Construido con Angular21 y un diseño basado en Tailwind + DaisyUI.

## Demo en producción


[Frontend desplegado en Vercel](https://lappiz-demo-frontend.vercel.app/)

[Backend relacionado](https://lappiz-demo-backend.onrender.com/)

## Stack

- Angular 21
- Tailwind CSS
- DaisyUI
- RxJS
- Cliente generado con [**ng-openapi-gen**](https://ng-openapi.dev/)

## Installation

1. Clona el repositorio:

```bash
$ git clone https://github.com/Alafresh/lappiz-demo-frontend.git
```

2. Instalar dependencias
```bash
$ npm install
```

3. Ejecuta en desarrollo
```bash
$ npm start
```
o
```bash
ng serve
```
La app quedará disponible en:
```bash
http://localhost:4200
```
